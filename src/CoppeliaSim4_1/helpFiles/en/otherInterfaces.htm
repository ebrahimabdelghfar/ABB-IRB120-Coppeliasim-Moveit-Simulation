Other Interfaces

The CoppeliaSim API framework can be extended in various ways, in order to offer interfaces that are not directly part of CoppeliaSim. This usually happens via a plugin, but other options are also available (add-ons, remote API clients, ROS nodes, BlueZero nodes, etc.). Following lists just a few of them:

  • OMPL plugin (path/motion planning).
  • Custom UI plugin (custom user interfaces)
  • CGAL plugin (surface reconstruction).
  • ICP plugin (point cloud matching).
  • Image plugin (OpenCV image processing).
  • Vision plugin (various image processing functions, also for special types of sensors).
  • Assimp plugin (CAD format import/export).
  • GLTF plugin (scene and animation export).
  • Kinematics plugin (forward/inverse kinematics, fully via API).
  • Geometric plugin (collision/distance calculation, fully via API).
