User config callback functions

A customization script can include a user config callback function, which is triggered when the user double-clicks the user parameter icon, in the scene hierarchy. This allows for a flexible behaviour for setting simulation parameters for instance: one can open a complex custom user interface for user interaction, or a simple file dialog as in following example:

function sysCall_userConfig()
    fileAndPath=sim.fileDialog(sim.filedlg_type_load,'Select file','','','text files','txt')
    print('Selected file is: ',fileAndPath)
end
